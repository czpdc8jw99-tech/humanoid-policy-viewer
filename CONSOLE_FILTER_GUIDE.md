# 控制台过滤器清除指南

## 问题：控制台隐藏了很多日志

控制台可能设置了过滤器，导致某些日志不显示。以下是清除方法：

---

## 方法1：清除控制台过滤器（Chrome/Edge）

### 步骤：
1. **打开控制台**（F12）
2. **找到过滤器输入框**（通常在控制台顶部，显示 "Filter" 或 "筛选"）
3. **点击过滤器输入框左侧的图标**（可能显示为漏斗图标或 "X"）
4. **或者直接清空过滤器输入框**（删除所有文字）
5. **确保所有日志级别都启用**：
   - ✅ Verbose（详细）
   - ✅ Info（信息）
   - ✅ Warnings（警告）
   - ✅ Errors（错误）

### 图示说明：
```
控制台顶部应该类似这样：
┌─────────────────────────────────────────┐
│ [🔍 Filter: ________] [🔽] [⚙️] [🚫]   │
│                                         │
│ ✅ Verbose  ✅ Info  ✅ Warnings  ✅ Errors │
└─────────────────────────────────────────┘
```

---

## 方法2：重置控制台设置

### Chrome/Edge：
1. 右键点击控制台
2. 选择 "Clear console"（清除控制台）
3. 或者按 `Ctrl+L` 清除日志
4. 检查过滤器设置是否被重置

---

## 方法3：检查日志级别设置

### Chrome DevTools：
1. 打开控制台（F12）
2. 点击控制台右上角的 **⚙️ 设置图标**
3. 找到 "Console" 或 "控制台" 设置
4. 确保以下选项都启用：
   - ✅ Show timestamps（显示时间戳）
   - ✅ Preserve log（保留日志）
   - ✅ Show CORS errors in console（显示CORS错误）

---

## 方法4：使用控制台命令清除过滤器

在控制台输入以下命令：

```javascript
// 清除所有过滤器
console.clear();

// 显示所有日志级别
console.log('=== 测试日志 ===');
console.info('=== 测试信息 ===');
console.warn('=== 测试警告 ===');
console.error('=== 测试错误 ===');
```

如果这些都能显示，说明过滤器已清除。

---

## 方法5：检查浏览器控制台设置

### Chrome：
1. 打开 DevTools（F12）
2. 点击右上角的 **⋮**（三个点）
3. 选择 **Settings**（设置）
4. 在左侧选择 **Console**
5. 检查以下设置：
   - ✅ **Show timestamps**（显示时间戳）
   - ✅ **Preserve log**（保留日志）
   - ✅ **Show CORS errors**（显示CORS错误）
   - ✅ **Hide network messages**（隐藏网络消息）- 可以取消勾选以显示更多日志

---

## 方法6：使用无痕模式测试

如果以上方法都不行，可以尝试：
1. 打开**无痕模式**（Ctrl+Shift+N）
2. 访问你的网页
3. 打开控制台
4. 查看是否能看到所有日志

---

## 快速检查：验证过滤器是否清除

运行以下代码，如果所有日志都能看到，说明过滤器已清除：

```javascript
console.log('%c这是 LOG 日志', 'color: blue;');
console.info('%c这是 INFO 信息', 'color: green;');
console.warn('%c这是 WARNING 警告', 'color: orange;');
console.error('%c这是 ERROR 错误', 'color: red;');
console.debug('%c这是 DEBUG 调试', 'color: purple;');
```

## 常见问题

### Q: 过滤器输入框在哪里？
**A:** 通常在控制台顶部，可能显示为：
- "Filter"（英文）
- "筛选"（中文）
- 一个搜索图标 🔍

### Q: 如何知道过滤器是否启用？
**A:** 如果过滤器输入框中有文字，或者某些日志级别被禁用（显示为灰色），说明过滤器已启用。

### Q: 清除后还是看不到某些日志？
**A:** 
1. 刷新页面（F5）
2. 重新加载策略
3. 检查是否有其他过滤器（例如网络过滤器、来源过滤器）

---

## 推荐操作流程

1. **清除控制台**：按 `Ctrl+L` 或点击清除按钮
2. **清除过滤器**：清空过滤器输入框
3. **启用所有日志级别**：确保 Verbose、Info、Warnings、Errors 都启用
4. **刷新页面**：按 `F5`
5. **重新加载策略**：选择 "G1 Locomotion (Gamepad)"
6. **运行诊断函数**：复制 `CONTROL_CONSOLE_STEPS.md` 中的诊断函数
